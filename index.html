LeagueFighters

by: Evan Mix

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because it was the only idea I had that I liked. Every other idea either sounded too hard, unoriginal, and/or boring

2. What I Learned

I learned from user testing what aspects of the game people did and didn't enjoy. I also learned about some minor bugs in my game

3. Architecture of My Game

My game has two main classes. One is the fighter class and the other is the GamePanel1 class. The fighter class has all the code for a base fighter and everything else is in the GamePanel1 class. I also have the LeagueFigher class which is the runner for my code

4. Technical Implementation

The thing I'm most happy about in my game is the animations I made for my fighters. While they may not be amazing they make the game for fun.

5. Most Challenging Part

I think the most challenging aspect of my game was coding the punching mechanics for my fighters. However, I also faced challenges creating the fighters and implementing the jumping physics

6. Future Fixes and Updates

If I were to continue working on this game I would probably improve the animations, add more fighters, and add a background. I could also probably add objects in the game or even maps
